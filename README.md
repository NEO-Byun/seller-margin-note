# 셀러마진 수첩

스마트스토어, 쿠팡, 오픈마켓 셀러가 판매가에서 실제로 남는 돈을 계산하는 정적 웹앱.

## 실행

`index.html`을 브라우저에서 열면 바로 동작.

서버, 계정, 데이터베이스 없음. 입력값은 브라우저 `localStorage`에 저장.

## 배포

정적 파일만 사용하므로 GitHub Pages, Netlify, Cloudflare Pages 어디든 올릴 수 있음.

현재 GitHub Pages 배포 완료.

```text
https://neo-byun.github.io/seller-margin-note/
```

## 기능

- 판매가·원가·수수료·배송비·광고비·반품률 입력
- 한 개 순이익, 순마진, 월 예상 이익 계산
- 손익분기 판매가와 목표 마진 판매가 계산
- 월 고정비 회수에 필요한 판매수 계산
- 상품별 저장, 비교, CSV 내보내기

## 왜 이걸 먼저 만듦

한국 타겟에서는 해외식 청구 흐름보다 `판매가에서 얼마 남는지`가 더 직관적.

온라인 판매자는 채널 수수료, 결제 수수료, 무료배송, 광고비, 반품비가 섞여 실제 이익을 놓치기 쉽다. 첫 버전은 회계툴이 아니라 가격 판단용 계산기로 둔다.

## 판매 실험

무료 계산기를 먼저 공개.

유료 상품은 작게 시작:

| 상품 | 가격 | 내용 |
|---|---:|---|
| 셀러 가격표 템플릿 | 19,000원 | 엑셀 템플릿, 상품군별 예시, 가격 점검표 |
| 마진 점검 키트 | 49,000원 | 쿠폰·광고비 시나리오, 시즌별 가격표, 상세페이지 문구 |
| 소형 셀러 Pro | 월 9,900원 | 여러 상품 저장, CSV import, 채널별 preset 관리 |

## 주의

채널 수수료는 상품군, 계약, 프로모션, 정산 방식에 따라 달라진다. 앱의 기본값은 시작점이며, 실제 정산 화면의 숫자로 바꿔서 계산해야 한다.
